## Installation
- install python
~~~
$ sudo apt-get update
$ sudo apt-get install python3.6
~~~
- create and activate virtual environment
~~~
$ pip install virtualenv
$ virtualenv venv
$ source venv/bin/activate
~~~
- install flask
~~~
$ sudo apt install python3-flask
~~~
- run application
~~~
$ flask run
~~~
